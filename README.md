# NewsClip - Aplikasi Kliping Berita Digital Personal

> **UAS Pemrograman Mobile** | Flutter + PHP (Vercel) + TiDB Cloud

## Fitur Utama
- Dashboard berita terkini (7 kategori)
- Pencarian berita global
- Simpan artikel sebagai kliping
- Kelola folder kliping (CRUD)
- Editor catatan & opini pribadi
- Status baca (Belum Dibaca / Selesai)
- Autentikasi JWT (Register & Login)

## Tech Stack
- **Frontend**: Flutter (Dart) + Provider + Dio
- **Backend**: PHP Serverless (Vercel)
- **Database**: TiDB Cloud (MySQL)
- **API**: NewsAPI.org

## Cara Menjalankan
```bash
git clone https://github.com/Afrilza218874/uas-pemrograman-mobile.git
cd uas-pemrograman-mobile
flutter pub get
flutter run
```
